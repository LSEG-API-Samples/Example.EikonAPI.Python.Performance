# Eikon Scripting API Samples

Following is a sampling of iPython notebooks to demonstrate how easy it is retrieve and plot the data from Eikon.

1. Compare Stock Performance - plots a spread chart of an equity and a benchmark
2. Dividends in equity chart - plots the overlay of dividend announcement dates on an equity chart
3. Stock Screener - plots a 2D and 3D bubble chart for constituents of an index.
